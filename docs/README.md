# BunnyEra Official Website Docs

This folder stores notes and planning for the BunnyEra public website (V1).

## Purpose

The website introduces BunnyEra to users, developers, partners, and community members.

## Homepage Sections (V1)

- Hero
- Company Overview
- Core Systems
- Business Directions
- Current Milestones
- Project Links / Ecosystem
- Contact / CTA

## Content Rules (Public-Safe)

Content must be:

- Clear and professional
- International and company-website tone
- Brand-consistent
- Free of secrets and internal-only details

Do not publish:

- API keys / tokens / `.env`
- Payment information
- Customer private data
- Internal credentials / private business documents
- Any restricted BunnyEra/Momo/IP character image assets

## Assets Policy

Public brand assets should be stored in `bunnyera-resources`.

If this website repository needs assets, only store approved public website copies under:

```txt
assets/icons/
assets/images/
```

## SEO Checklist (V1)

- Title includes “BunnyEra”
- Meta description clearly states: AI-driven cross-border technology company
- Canonical: `https://bunnyera.com/`
- Open Graph meta tags
- Twitter card meta tags
- `robots.txt` allows crawling
- `sitemap.xml` contains `https://bunnyera.com/`
